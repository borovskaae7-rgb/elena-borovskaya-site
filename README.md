# Лендинг Елены Боровской

Статический лендинг специалиста по VK Ads и автоворонкам. Сайт можно открыть локально без сборки или разместить на любом статическом хостинге.

## Локальный запуск

```bash
python3 -m http.server 4173
```

После запуска откройте `http://localhost:4173`.

## Перед публикацией

1. Замените ссылки `https://t.me/username` на ссылку с актуальным Telegram username.
2. Добавьте коды Яндекс Метрики и VK Pixel в отмеченные комментариями места в `<head>` файла `index.html`.
3. Дополните политику конфиденциальности реквизитами оператора персональных данных и проверьте её с юристом с учётом подключённых сервисов.

## Страницы

- `index.html` — основная страница лендинга.
- `privacy.html` — политика конфиденциальности.
- `thank-you.html` — страница благодарности после отправки заявки.
